Accompanying code to [Optimal energy and redox metabolism in the cyanobacterium Synechocystis sp. PCC 6803](https://www.nature.com/articles/s41540-023-00307-3).

### Python package versions:

* Python 3.8.5
* Pickle 4.0
* Numpy 1.19.4
* Pandas 1.3.0
* Matplotlib 3.3.3 
* COBRApy 0.20.0
* MEMOTE 0.12.0
